Detailed Description

template<typename T>
class fsm::FSM< T >

FSM is a class that implments a finite state machine. An FSM instance can be in an exactly one state at a time from a finite number of possible states. Changes from state to state happen by reading an input symbol from the alphabet of the FSM and follow the transiton rules defined by the transition_table of the FSM.
